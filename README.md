# React Functional Component Project

This project is a **React application** built using **functional components** and **Vite** as the build tool. It demonstrates modern React practices, including hooks and component-based architecture.

## Features

- Built with **React Functional Components**.
- Uses **React Hooks** like `useState` and `useEffect`.
- Component-based structure for **reusability and maintainability**.
- Fast setup with **Vite**.
- Demonstrates state management within functional components.
- Easy to extend for larger projects.

````

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Naol724/React-functional-component.git
````

2. Navigate into the project folder:

```bash
cd React-functional-component
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

Open your browser at the URL shown in the terminal (usually `http://localhost:5173`) to view the app.

## Usage

* Modify or create new functional components inside `src/components/`.
* Use **props** and **state** to manage data between components.
* Experiment with **React Hooks** for dynamic behavior.
